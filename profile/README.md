# ExpansionPak


[![image](https://avatars.githubusercontent.com/u/271502946?s=400&u=ae83108f7c426d94cedc804ac687f9c48e117173&v=4)](https://github.com/ExpansionPak)

**Recompiled. Reimagined. Rebuilt.**

ExpansionPak is a collection of low-level tools focused on bringing retro games into the modern era through **static recompilation**, **reverse engineering**, and **runtime experimentation**.

## What we do

- Static recompilers for retro consoles  
- Native ports from original game binaries  
- Reverse engineering frameworks  
- Graphics and runtime bridges  
- Experimental tooling pushing beyond traditional emulation  

---

## Projects

### GCRecompiler
GameCube recompilation experiments.  
Exploring PowerPC → native pipelines.

### gx2gl
Graphics translation layer bridging console GPU pipelines into modern APIs.

---

## Status

Active development is focused on newer targets and experimental pipelines.  
Older projects may be archived or transferred as the direction evolves.
